# AED
📚 Algoritmos e Estruturas de Dados em C

Repositório dedicado ao estudo e implementação de algoritmos fundamentais e estruturas de dados utilizando a linguagem C.

O objetivo deste projeto é consolidar conhecimentos em lógica de programação, análise de complexidade e estruturas de dados clássicas, que são a base para áreas como desenvolvimento de software, ciência de dados e engenharia de sistemas.

🎯 Objetivos do Repositório

Praticar implementação de algoritmos clássicos

Aprender estruturas de dados fundamentais

Desenvolver raciocínio algorítmico

Entender análise de complexidade (Big O)

Criar uma base sólida para entrevistas técnicas e estudos avançados
